1. Phép nội suy WENO¶

Xt li u $x_i = i\Delta x$. Gi s gi tr hm u(x) ti mi im li bit: $u_i = u(x_i)$ Khi gi tr xp x ca u(x) ti mt im bt k, v d ti $x_{i+1/2}$, c th xc nh c bng php ni suy. C th, trong khun (stencil) $S_1 = \{x_{i-2}, x_{i-1}, x_i\}$, ti 3 im li $i-2, i-1, i$ cho gi tr $u_{i-2}, u_{i-1}, u_i$. Cn tm a thc ni suy $p_1(x)$ tha mn: $p_1(x_{i-2}) = u_{i-2},\ p_1(x_{i-1}) = u_{i-1},\ p_1(x_i) = u_i$. Khi ta c th ly xp x $u_{i+1/2}^{(1)} = p_1(x_{i+1/2})$.

S dng a thc ni suy Lagrange, bin i ta c $$u_{i+1/2}^{(1)} = \frac{3}{8}u_{i-2} - \frac{5}{4}u_{i-1}+\frac{15}{8}u_i \qquad (1)$$ Xp x ny c chnh xc bc ba nu hm u(x) trn trong khun $S_1$: $$u_{i+1/2}^{(1)} - u(x_{i+1/2} = O(\Delta x^3)$$ Tng t i vi khun $S_2 = \{x_{i-1}, x_{i}, x_{i+1}\}$ ta c: $$u_{i+1/2}^{(2)} = -\frac{1}{8}u_{i-1} + \frac{3}{4}u_{i}+\frac{3}{8}u_{i+1} \qquad (2)$$ khun $S_3 = \{x_{i}, x_{i+1}, x_{i+2}\}$: $$u_{i+1/2}^{(3)} = \frac{3}{8}u_{i} + \frac{3}{4}u_{i+1}-\frac{1}{8}u_{i+2} \qquad (3)$$

Nu s dng khun ln hn $S = \{x_{i-2}, x_{i-1}, x_{i}, x_{i+1}, x_{i+2}\}$ ta s thu c xp x c chnh xc bc 5:

$u_{i+1/2} = \frac{3}{128}u_{i-2} - \frac{5}{32}u_{i-1}+\frac{45}{64}u_{i} +\frac{15}{32}u_{i+1}-\frac{5}{128}u_{i+2} \qquad (4)$$

Vi php ni suy WENO cng thc xp x (4) bc 5 trn khun ln S c th thu c bng t hp tuyn tnh 3 xp x bc 3 $u_{i+1/2}^{(1)}, u_{i+1/2}^{(2)}, u_{i+1/2}^{(3)}$ trn 3 khun nh $S_1, S_2, S_3$:

$$u_{i+1/2} = \gamma_1 u_{i+1/2}^{(1)} + \gamma_2 u_{i+1/2}^{(2)} +\gamma_3 u_{i+1/2}^{(3)}$$

vi cc hng s $\gamma$ tha mn $\gamma_1+\gamma_2+\gamma_3 = 1$ thng c gi l *trng lng tuyn tnh (linear weights)*, trong trng hp ny: $$\gamma_1 = 1/16;\ \gamma_2 = 5/8;\ \gamma_3 = 5/16$$

Nh vy nu hm u(x) trn trn khun S th ta s c s bc 5 v c ba xp x u s dng c. Trng hp xut hin im gin on trong S th lc ny khng phi c ba xp x u tt. Phng php ENO c in s la chn mt trong ba xp x cn c vo trn ca hm u(x) trong ba khun nh. Vi cch ti im gin on ta vn c li gii vi chnh xc bc ba v m bo khng xut hin dao ng nu u(x) trn trong t nht mt khun nh.

S WENO s dng xp x l t hp ba xp x bc ba: $$u_{i+1/2} = w_1 u_{i+1/2}^{(1)} + w_2 u_{i+1/2}^{(2)} +w_3 u_{i+1/2}^{(3)}$$ vi $w_j \geq 0, w_1+w_2+w_3=1$ gi l trng lng phi tuyn v tha mn iu kin:

  • $w_j \approx \gamma_j$ if u(x) trn trong khun ln S
  • $w_j \approx 0$ nu u(x) gin on trn khun $S_j$ nhng trn trong t nht mt trong hai khun cn li.

Trng lng $w_j$ c xc nh cn c vo ch th trn (smoothness indicator) $\beta_j$ trong khun $S_j$. Nu ch th trn cng ln th hm u(x) cng t trn.

Mt trong nhng cng thc xc nh ch th trn l: \begin{array}{l} \beta_1 = \frac{1}{3}\left(4u_{i-2}^2 - 19u_{i-2}u_{i-1} + 25u_{i-1}^2 + 11u_{i-2}u_i - 31u_{i-1}u_i + 10u_i^2 \right)\\ \beta_2 = \frac{1}{3}\left(4u_{i-1}^2 - 13u_{i-1}u_{i} + 13u_{i}^2 + 5u_{i-1}u_{i+1} - 13u_{i}u_{i+1} + 4u_{i+1}^2 \right)\\ \beta_3 = \frac{1}{3}\left(10u_{i-1}^2 - 31u_{i}u_{i+1} + 25u_{i+1}^2 + 11u_{i}u_{i+2} - 19u_{i+1}u_{i+2} + 4u_{i+2}^2\right) \end{array} Khi , trng lng phi tuyn xc nh nh sau:

$w_j = \frac{\tilde w_j}{\tilde w_1+\tilde w_2+\tilde w_3}$ vi $\tilde w_j=\frac{\gamma_j}{(\varepsilon + \beta_j)^2}$

y $\varepsilon$ c th ly bng 1e-6 nhm trnh chia cho 0.

2. Tái cấu trúc WENO¶

So vi phng php ni suy WENO, phng php ti cu trc WENO ph hp hn gii cc nh lut bo ton. Trong phng php ny, thay v s dng gi tr hm u(x) ti cc im li ta dng gi tr trung bnh trong th tch hu hn $I_i=(x_{i-1/2}, x_{i+1/2})$: $$\overline{u_i} = \frac{1}{\Delta x}\int_{x_{i-1/2}}^{x_{i+1/2}} u(x) dx$$

Cng nh phn trc, iu chng ta mun l tm gi tr hm u(x) ti mt im cho trc v d ti $x_{i+1/2}$.

Nu ta nh ngha mt nguyn hm ca u(x): $$U(x) = \int_{x_{-1/2}}^x u(\xi) d\xi$$ vi cn di $x_{-1/2}$ c th thay bi bt k im c nh no th ta c:

$$U(x_{i+1/2} = \int_{x_{-1/2}}^{x_{i+1/2}} u(\xi) d\xi = \sum_{l=0}^i \int_{x_{l-1/2}}^{x_{l+1/2}} u(\xi) d\xi = \sum_{l=0}^i \Delta x \overline{u_l}$$

Nh vy, nu bit gi tr trung bnh u trong mi th tch th ta s bit gi tr nguyn hm ti cc mt th tch $U(x_{i+1/2})$. Gi tr u c th ly xp x $u(x) = U'(x)$. Gi thit $P_1(x)$ l a thc ni suy bc 3 hm U(x) ti 4 im $x_{j+1/2}, j=i-3, i-2, i-1, i$ v $p_1(x)=P'_1(x)$, khi $p_1(x)$ l a thc bc hai ti cu trc hm u(x) trong khun $S_1=\{I_{i-2}, I_{i-1}, I_i\}$ vi: $$(\overline{p_1})_j = \frac{1}{\Delta x}\int_{x_{j-1/2}}^{x_{j+1/2}} p_1(x) dx = \overline{u_j}, j=i-2, i-1, i$$

Gi tr $u(x_{i+1/2})$ c th ly xp x $u_{i+1/2}^{(1)} \equiv p_1(x_{i+1/2})$. Cc php bin i i s cho ta cng thc xp x bc 3: $$u_{i+1/2}^{(1)} = \frac{1}{3}\bar{u}_{i-2} - \frac{7}{6}\bar{u}_{i-1} + \frac{11}{6}\bar{u}_{i} $$ Tng t cho cc khun $S_2, S_3$ ta c: $$u_{i+1/2}^{(2)} = -\frac{1}{6}\bar{u}_{i-1} + \frac{5}{6}\bar{u}_{i} + \frac{1}{3}\bar{u}_{i+1} $$ $$u_{i+1/2}^{(3)} = \frac{1}{3}\bar{u}_{i} + \frac{5}{6}\bar{u}_{i+1} - \frac{1}{6}\bar{u}_{i+2} $$

Nu s dng khun ln $S=\{I_{i-2}, I_{i-1}, I_i, I_{i+1},I_{i+2}\}$ ta c xp x bc 5: $$u_{i+1/2} = \frac{1}{30}\bar{u}_{i-2} - \frac{13}{60}\bar{u}_{i-1} + \frac{47}{60}\bar{u}_{i} + \frac{9}{20}\bar{u}_{i+1} - \frac{1}{20}\bar{u}_{i+2}$$

Tng nh phn trc ta c: $$u_{i+1/2} = \gamma_1 u_{i+1/2}^{(1)} + \gamma_2 u_{i+1/2}^{(2)} +\gamma_3 u_{i+1/2}^{(3)}$$ vi cc hng s: $$\gamma_1 = 1/10;\ \gamma_2 = 3/5;\ \gamma_3 = 3/10$$

Xp x cui cng c chn: $$u_{i+1/2} = w_1 u_{i+1/2}^{(1)} + w_2 u_{i+1/2}^{(2)} +w_3 u_{i+1/2}^{(3)}$$

Vi ch th trn:

\begin{array}{l} \beta_1 = \frac{13}{12}\left(\bar{u}_{i-2} - 2\bar{u}_{i-1}+\bar{u}_{i}\right)^2 + \frac{1}{4}\left(\bar{u}_{i-2} - 4\bar{u}_{i-1}+3\bar{u}_{i}\right)^2\\ \beta_2 = \frac{13}{12}\left(\bar{u}_{i-1} - 2\bar{u}_{i}+\bar{u}_{i+1}\right)^2 + \frac{1}{4}\left(\bar{u}_{i-1} - \bar{u}_{i+1}\right)^2\\ \beta_3 = \frac{13}{12}\left(\bar{u}_{i} - 2\bar{u}_{i+1}+\bar{u}_{i+2}\right)^2 + \frac{1}{4}\left(3\bar{u}_{i} - 4\bar{u}_{i+1}+\bar{u}_{i+2}\right)^2 \end{array}

S dng tnh cht i xng ta c cng thc xc nh gi tr $u_{i-1/2}$ nh sau: $$u_{i-1/2}^{(1)} = -\frac{1}{6}\bar{u}_{i-2} + \frac{5}{6}\bar{u}_{i-1} + \frac{1}{3}\bar{u}_{i} $$ $$u_{i-1/2}^{(2)} = \frac{1}{3}\bar{u}_{i-1} + \frac{5}{6}\bar{u}_{i} - \frac{1}{6}\bar{u}_{i+1} $$ $$u_{i-1/2}^{(3)} = \frac{11}{6}\bar{u}_{i} - \frac{7}{6}\bar{u}_{i+1} + \frac{1}{3}\bar{u}_{i+2} $$

$$\gamma_1 = 3/10;\ \gamma_2 = 3/5;\ \gamma_3 = 1/10$$

3. Sơ đồ sai phân phương pháp thể tích hữu hạn¶

Xt phng trnh Burgers khng nht: $$\frac {\partial u}{\partial t} + \frac{\partial F}{\partial x} = 0; \ F=\frac{u^2}{2} \qquad (1)$$

vi s sai phn phng php th tch hu hn: $$\frac {d\bar{u}}{dt} + \frac{F_{i+1/2} - F_{i-1/2}}{\Delta x} = 0 \qquad(2)$$

Cng ging nh phng php Godunov hay MUSCL trong bi 12, 13 chng ta s gii phng trnh Burgers theo ba bc:

    1. s dng xp x bc 5 WENO ti cu trc nghim trong tng cell, tm $u_{i+1/2}^L, u_{i+1/2}^R$. Gi tr uL bn tri c xc nh bng xp x trong khun $S_L = \{I_{i-2}, I_{i-1}, I_{i}, I_{i+1}, I_{i+2}\}$. Tng ng uR - $S_R = \{I_{i-1}, I_{i}, I_{i+1}, I_{i+2}, I_{i+3}\}$. Nh vy ta c gi tr $u_{i+1/2}^L = u_{i+1/2}, u_{i+1/2}^R = u_{(i+1)-1/2}$.

    1. tm nghim bi ton phn r gin on trn b mt $i+1/2$, tnh hm dng.

    1. tm nghim u bc thi gian tip theo bng phng php Runge-Kutta bc ba:

      • vit li phng trnh Burgers dng: $$\frac {\partial u}{\partial t} = L(u)$$ vi ton t $L(u) = -\frac{\partial F}{\partial x}$.

      • ri rc thi gian theo ba bc: \begin{array}{l} \bar{u}^{(1)} = \bar{u}^n + \Delta t L(\bar{u}^n)\\ \bar{u}^{(2)} = \frac{3}{4}\bar{u}^n + \frac{1} {4}\bar{u}^{(1)} + \frac{1}{4} \Delta t L(\bar{u}^{(1)})\\ \bar{u}^{n+1} = \frac{1}{3}\bar{u}^n + \frac{2}{3}\bar{u}^{(2)} + \frac{2}{3} \Delta t L(\bar{u}^{(2)}) \end{array}

Bi ton

Nh hai bi trc, ta xt hai bi ton lan truyn sng xung kch v sng phn r trong thi gian dao ng cn cha lan truyn ti bin. Xem nh iu kin trn bin u khng i. Vi cc cng thc tnh xp x u, ta cn 2 trn bin tri v 3 trn bin phi. Tc vng tnh ton: $x[2:-3]$.

import numpy as np
import matplotlib.pyplot as plt


#WENO5 Interpolation: smothness indicators
def beta_Itp(a, u):
    b = 1./3*(a[0]*u[0]**2 + a[1]*u[0]*u[1] + a[2]*u[1]**2 + a[3]*u[0]*u[2] + a[4]*u[1]*u[2] + a[5]*u[2]**2)
    return b

#WENO5 Reconstruction: smothness indicators
def beta_Rct(a, u):
    b = 13./12*(u[0] - 2*u[1] + u[2])**2 + 0.25*(a[0]*u[0] + a[1]*u[1] + a[2]*u[2])**2
    return b

#Hm tm trng lng phi tuyn 
def omega(gamma, beta):
    omgs = gamma/(epsilon + beta)**2
    omgs_sum = sum(omgs)
    omg = omgs/omgs_sum
    return omg

#Hm ti cu trc nghim
def reconstruction(u, beta, a_b, gamma, a_u, u_05):
    for i in range(2, len(u)-2): #for each cell in cells
        #bc 1. tnh ch th  trn trong tng khun S_j
        b0 = beta(a_b[0], u[i-2:i+1])
        b1 = beta(a_b[1], u[i-1:i+2])
        b2 = beta(a_b[2], u[i  :i+3])
        
        #bc 2. tm trng lng phi tuyn tng khun
        omg = omega(gamma, np.array([b0, b1, b2]))
        
        #bc 3. tm xp x bc 3
        u0 = sum(a_u[0] * u[i-2:i+1])
        u1 = sum(a_u[1] * u[i-1:i+2])
        u2 = sum(a_u[2] * u[i  :i+3])
        
        #bc 4. tm xp x bc 5
        u_05[i] = sum(omg * [u0, u1, u2])
        
        #iu kin bin: u_05[0], u_05[1], u_05[-2], u_05[-1]
    return

#Phn r gin on godunov trn tng mt th tch hu hn 
def decay_godunov(ul, ur): #uL, uR l hai s
    ustar = 0
    if ul > ur:
        if (ul + ur)/2. > 0: ustar = ul
        else: ustar = ur
    else:
        if ul > 0: ustar = ul
        elif ur < 0: ustar = ur
        else: ustar = 0
    return ustar

def flux(u):
    return u**2/2.

#solver
def solver(u, beta, a_b, gamma_p05, gamma_m05, a_up05, a_um05, nt):
    u_p05 = u.copy()
    u_m05 = u.copy()
    un   = u.copy()
    for n in range(nt):
        un = u.copy()
        for stage in range(3): #runge-kutta 3
            #Step 1: reconstruction u_{i+1/2}, u_{i-1/2}
            reconstruction(u, beta, a_b, gamma_p05, a_up05, u_p05)
            reconstruction(u, beta, a_b, gamma_m05, a_um05, u_m05)

            #Step 2: riemann solver
            ustar = np.array([decay_godunov(ul, ur) for ul, ur in zip(u_p05[1:-2], u_m05[2:-1])])

            #Step 3: time intergration
            if stage == 0: u[2:-2] = un[2:-2] - dt/dx*(flux(ustar[1:]) - flux(ustar[:-1]))
            elif stage == 1: u[2:-2] = 0.75*un[2:-2] + 0.25*u[2:-2] - 0.25*dt/dx*(flux(ustar[1:]) - flux(ustar[:-1]))
            else: u[2:-2] = 1./3*un[2:-2] + 2./3*u[2:-2] - 2./3*dt/dx*(flux(ustar[1:]) - flux(ustar[:-1]))
            #iu kin bin u[0], u[1], u[-2], u[-1]
    return
