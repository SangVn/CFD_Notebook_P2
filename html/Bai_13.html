Bài 13. Định lý Godunov, sơ đồ TVD, sơ đồ MUSCL¶

Xt phng trnh Burgers khng nht: $$\frac {\partial u}{\partial t} + \frac{\partial F}{\partial x} = 0; \ F=\frac{u^2}{2} \qquad (1)$$

vi s sai phn phng php th tch hu hn: $$\frac{u_i^{n+1} - u_i^n}{\Delta t} + \frac{F_{i+1/2} - F_{i-1/2}}{\Delta x} = 0 \qquad(2)$$

1. Định lý Godunov¶

nh l Godunov pht biu v tnh n iu ca s sai phn. S sai phn gi l n iu (monotone scheme) nu nh vi hai iu kin ban u $u^0 > v^0$ th $u^n > v^n$. Nu s sai phn cho nghim $u^n$ n iu tng (gim) v $u^{n+1}$ cng n iu tng t th c ngha l tnh n iu c duy tr (monotonicity preserving).

Godunov ch ra rng: s sai phn n iu c bc khng qu bc mt.

Thc vy, bi 2 v bi 12 cho thy, s bc hai lm xut hin cc dao ng khi nghim gin on, tc l iu kin n iu khng c tha mn, s cho nghim khng chnh xc. Tuy nhin so vi s bc mt, s bc hai cho nghim chnh xc hn nu nghim trn (mt), ti ni nghim gin on s bc hai tuy lm xut hin cc dao ng, nhng dc gin on ln hn. S l mt s tt nu kt hp c hai s trn. tng l ta s s dng mt 'cng tc chuyn i' gia s bc 2 v bc 1 ty vo nghim mt hay gin on.

2. Sơ đồ TVD (total variation dimishing)¶

Lax (1973) ch ra rng, nh lut bo ton mt i lng v hng c m t bng phng trnh Burgers, c bin phn ton phn (total variation) ca nghim vt l khng tng theo thi gian. Vi bin phn ton phn (TV) l: $$TV = \int |\partial u/\partial x| dx$$ i vi nghim ri rc: $$TV(u) = \sum_i|u_{i+1}-u_i|$$ Mt s sai phn c gi l TVD nu nh: $$TV(u^{n+1}) \leq TV(u^n)$$

Harten (1983) chng minh:

  1. s n iu th TVD
  2. s TVD th duy tr tnh n iu (monotonicity-preserving: nu $u_{i+1}^n > u_i^n$ th $u_{i+1}^{n+1} > u_i^{n+1} \forall i$ )

Chng ta c th xy dng s TVD bng phng php MUSCL.

3. Sơ đồ MUSCL (Monotonic Upwind Scheme for Conservation Laws)¶

Quay tr li vi phng php Godunov bi 12, hm dng F trn b mt th tch c tnh nh sau: $$F_{i+1/2} = F(u_{i+1/2}^*(u_{i+1/2}^L, u_{i+1/2}^R))$$ vi $u^*$ l nghim bi ton Riemann $u^*(u_L, u_R)$. Phng php Godunov c th chia l ba bc:

  • Bc 1: Ti cu trc nghim (reconstruction) trong tng th tch hu hn. Thc cht l xc nh gi tr uL, uR hai bn b mt tm $u^*$ trn b mt th tch.
  • Bc 2: Gii bi ton Riemann v phn r gin on trn b mt th tch $u^*(u_L, u_R)$ tm $u^*$, tnh dng F.
  • Bc 3: Tm nghim bc thi gian tip theo $u^{n+1}$

bc 1, tm $u_{i+1/2}^*(u_{i+1/2}^L, u_{i+1/2}^R)$ Godunov xut cho cc hng s: $u_{i+1/2}^L = u_i, u_{i+1/2}^R = u_{i+1}$. Cch ny gi l ti cu trc hng s tng mnh (piecewise constant). Ngoi ra cn c ti cu trc tuyn tnh tng mnh (piecewise linear) v ti cu trc parabolic tng mnh (piecewise parabolic) (th t tng ng nh trn hnh):

V d ti cu trc tuyn tnh tng mnh, s dng khai trin Taylor: $$ u_{i+1/2} = u_i + \frac{\partial u}{\partial x} (x_{i+1/2} - x_i)\\ u_{i+1/2} = u_i + \frac{u_i - u_{i-1}}{\Delta x} \frac{\Delta x}{2}\\ u_{i+1/2} = u_i + 0.5(u_i - u_{i-1}) \qquad (3) $$ Cng thc (3) nn s dng khi nghim mt m, khi nghim gin on ta ch cn ly $u_{i+1/2} = u_i$. Ta c th thm vo (3) cng tc chuyn i $\phi: \ u_{i+1/2} = u_i + 0.5\phi(u_i - u_{i-1})$, vi $\phi$ bng 0 khi nghim gin on, bng 1 khi nghim mt m.

Trn tng , ra i s MUSCL dng ti cu trc tuyn tnh tng mnh v s dng 'cng tc chuyn i' gi l gii hn dc (slope limiter):

$$ u_{i+1/2}^L = u_i + 0.5 \phi(r_i)(u_{i+1}-u_i);\ u_{i+1/2}^R = u_{i+1} - 0.5 \phi(r_{i+1})(u_{i+2}-u_{i+1})\\ u_{i-1/2}^L = u_{i-1} + 0.5 \phi(r_{i-1})(u_i-u_{i-1});\ u_{i-1/2}^R = u_i - 0.5 \phi(r_i)(u_{i+1}-u_i)\\ r_i = \frac{u_i - u_{i-1}}{u_{i+1} - u_{i}} \qquad (4) $$

Hm $\phi(r)$ chnh l gii hn dc. C nhiu hm gii hn khc nhau tham kho:

minmod - (Roe, 1986):

$$\phi_{mm}(r) = max[0, min(1, r)]$$

superbee - (Roe, 1986):

$$\phi_{sb}(r) = max[0, min(2r,1), min(r, 2)]$$

van leer - (van leer 1974):

$$\phi_{vl}(r) = \frac{r+|r|}{1+|r|}$$

Koren - (Koren 1993)

$$\phi_{kr} = max\left[0, min\left(2r, min\left(\frac{1+2r}{3}, 2\right)\right)\right]$$

Hm gii hn bng khng khi r<0 v bng 1 khi r=1, iu ny m bo rng ti nhng ni nghim gin on, s s chuyn v bc mt, nhng v tng th ta c s bc hai.

Bài toán¶

Xt bi lan truyn sng xung kch vi iu kin ban u nh bi 12: $u[0.<x<0.5] = 1.2; u[0.5<x<1.] = 0.4$

import numpy as np
import matplotlib.pyplot as plt

#chia li trn on [0, 1]
nx = 41
x, dx = np.linspace(0, 1, nx, retstep=True)

#bc thi gian
r = 0.4
dt = r*dx

#iu kin ban u
uL = 1.2
uR = 0.4

u0 = np.zeros(nx)
i = int(0.5/dx)
u0[:i] = uL
u0[i:] = uR

#Phn r gin on godunov trn tng mt th tch hu hn 
def decay_godunov(uL, uR):
    ustar = 0
    if uL > uR:
        if (uL + uR)/2. > 0: ustar = uL
        else: ustar = uR
    else:
        if uL > 0: ustar = uL
        elif uR < 0: ustar = uR
        else: ustar = 0
    return ustar


#tnh hm dng F(u*)
def flux(u):
    return u**2/2.

#Ta c th vit code s  muscl nh sau y, nhng cch ny chm, tn b nh
def muscl(limiter, nt):
    u = u0.copy()
    un = np.zeros(nx)
    for n in range(nt):
        un = u.copy()
        for i in range(2,nx-2):
            r_im1 = (un[i-1] - un[i-2])/(un[i] - un[i-1] + 1e-6)
            r_i   = (un[i] - un[i-1])/(un[i+1] - un[i] + 1e-6)
            r_ip1 = (un[i+1] - un[i])/(un[i+2] - un[i+1] + 1e-6)
            
            u_im05L = un[i-1] + 0.5*limiter(r_im1)*(un[i] - un[i-1])
            u_im05R = un[i] - 0.5*limiter(r_i)*(un[i+1] - un[i])
            u_im05 = decay_godunov(u_im05L, u_im05R) #u_{i-1/2}
            
            u_ip05L = un[i] + 0.5*limiter(r_i)*(un[i+1] - un[i])
            u_ip05R = un[i+1] - 0.5*limiter(r_ip1)*(un[i+2] - un[i+1])
            u_ip05 = decay_godunov(u_ip05L, u_ip05R) #u_{i+1/2}
            
            u[i] = un[i] - dt/dx*(flux(u_ip05) - flux(u_im05))
            
    return u

# vit li on code nh sau 
def MUSCL_scheme(limiter, nt):
    u = u0.copy()
    r = np.zeros(nx)
    uL = np.zeros(nx)
    uR = np.zeros(nx)
    for n in range(nt):
        #tm r,  trnh chia cho 0 ta cng vo mu s 1e-6
        r[1:-1] = (u[1:-1] - u[:-2])/((u[2:] - u[1:-1]) + 1e-6)
        
        #ti cu trc nghim u_{i+1/2}^L
        uL[1:-2] = u[1:-2] + 0.5*limiter(r[1:-2])*(u[2:-1] - u[1:-2])
        
        #ti cu trc nghim u_{i+1/2}^R
        uR[1:-2] = u[2:-1] - 0.5*limiter(r[2:-1])*(u[3:] - u[2:-1])
        
        #gii bi ton riemann 
        ustar = np.array([decay_godunov(ul, ur) for ul, ur in zip(uL[1:-2], uR[1:-2])])

        #tnh hm dng v tm nghim u  bc thi gian tip theo 
        u[2:-2] = u[2:-2] - dt/dx*(flux(ustar[1:]) - flux(ustar[:-1]))
        
        #iu kin bin khng i 
    return u


#slope limiter: phi(r), rs - mng r
def minmod(rs):
    phi = np.array([max(0, min(1,r)) for r in rs])
    return phi

def koren(rs):
    phi = np.array([max(0, min(2*r, min((1+2*r)/3, 2))) for r in rs])
    return phi

def superbee(rs):
    phi = np.array([max(0, min(2*r, 1), min(r,2)) for r in rs])
    return phi

def vanleer(rs):
    phi = np.array([(r+abs(r))/(1+abs(r)) for r in rs])
    return phi

#tm nghim
nt = 25
u_mm = MUSCL_scheme(minmod, nt)
u_kr = MUSCL_scheme(koren, nt)
u_sp = MUSCL_scheme(superbee, nt)
u_vl = MUSCL_scheme(vanleer, nt)

#nghim chnh xc: vn tc lan truyn sng xung kch s=(uL+uR)/2. = 0.8
u_exact = np.zeros(nx)
i = int((0.5 + 0.8*nt*dt)/dx)
u_exact[:i] = uL
u_exact[i:] = uR
