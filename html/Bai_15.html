Bài 15. So sánh sơ đồ Godunov, MUSCL, WENO5¶

Nh vy ta thc hnh gii phng trnh Burgers vi ba s Godunov bc 1, MUSCL bc hai v WENO5 bc 5. bi ny trc ht cc bn hy tng hp ba s trn vo mt module gi l burgers_solver cha trong th mc lib. gi module burgers_solver, ta cn to mt file rng vi tn l $__init__.py$ trong lib.

By gi ta s s dng module ny gii bi ton Zalesak vi iu kin ban u l bn nh thp lin tip: nhn, vung, nhn, t.

#import th vin, module 
import matplotlib.pyplot as plt
from lib.burgers_solver import*

#li v thng s tnh ton 
nx = 81
x, dx = np.linspace(-1, 1, nx, retstep=True)
r = 0.4
dt = r*dx
nt = 15

#Bi ton Zalesak 
a = 0.5
z = -0.7
delta = 0.005
alpha = 10.
beta = np.log(2.)/(36*delta**2)

def G(x, z):
    return np.exp(-beta*(x-z)**2)

def F(x, a):
    return np.sqrt(abs(1-alpha**2*(x-a)**2))

#iu kin ban u
u0 = np.zeros(nx)
for i, xi in enumerate(x):
    if (-0.8 <= xi <= -0.6): u0[i] = 1./6*(G(xi, z-delta) + G(xi, z+delta) + 4*G(xi, z))
    elif (-0.4 <= xi <= -0.2): u0[i] = 1.
    elif (0 <= xi <= 0.2): u0[i] = 1 - abs(10*(xi-0.1))
    elif (0.4 <= xi <= 0.6): u0[i] = 1./6*(F(xi, a-delta) + F(xi, a+delta) + 4*F(xi,a))

#xt phng trnh chuyn dch tuyn tnh du/dt + du/dx = 0
def flux(u):
    return u

#nghim chnh xc        
i = int(nt*dt/dx)
u_exact = np.roll(u0, i) #dch chuyn sang phi i bc 

#trc ht ta so snh cc gii hn  dc trong MUSCL 
u_mm = u0.copy()
u_kr = u0.copy()
u_sb = u0.copy()
u_op = u0.copy()
u_vl74 = u0.copy()
u_vl77 = u0.copy()

muscl(u_mm, minmod, flux, dx, dt, nt)
muscl(u_kr, koren, flux, dx, dt, nt)
muscl(u_sb, superbee, flux, dx, dt, nt)
muscl(u_op, ospre, flux, dx, dt, nt)
muscl(u_vl74, vanleer74, flux, dx, dt, nt)
muscl(u_vl77, vanleer77, flux, dx, dt, nt)


fig = plt.figure(figsize=[16, 10])
plt.plot(x, u_exact, 'k', x, u_mm, 'c', x, u_kr, 'r+', x, u_sb, 'go',)
plt.plot(x, u_op, 'bx', x, u_vl74, 'y', x, u_vl77, 'm')
plt.legend(['u_exact', 'minmod', 'koren', 'superbee', 'ospre', 'vanleer74', 'vanleer77'])
plt.title('Zalesak problem')
plt.show()

Vi 4 nh thp lin tip: nhn - vung - nhn - t, cc s phn no bc l kh nng ca mnh. Tuy nhin kh l kh a ra kt lun cui cng ci no tt nht. Xt theo cao v hnh dng thp, ta c th la chn s ospre hoc vanleer74 so snh vi cc phng php cn li.

Ta thy s Godunov cho kt qu km nht. Hai s WENO5 cho kt qu nh nhau. Kt qu thu c bng phng php WENO5 v MUSCL kh ging nhau. S MUSCL chim u th ti cc chn thp v nh thp vung - nhng v tr gin on, cn WENO5 chim nh thp nhn v t - v tr cc i. Tip theo ta gii bi ton chuyn dch khng tuyn tnh.
