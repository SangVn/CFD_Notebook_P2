Bài 12. Sơ đồ Godunov, bài toán Riemann¶

1. Sơ đồ sai phân bảo toàn¶

Xt phng trnh Burgers khng nht dng khng phn k: $$\frac {\partial u}{\partial t} + u\frac{\partial u}{\partial x} = 0 \qquad (1)$$ Phng trnh ny c th biu din li dng phn k: $$\frac {\partial u}{\partial t} + \frac{\partial F}{\partial x} = 0; \ F=\frac{u^2}{2} \qquad (2)$$

S dng phng php sai phn hu hn vi s upwind ta c hai s sai phn tng ng dng khng bo ton v bo ton: $$\frac{u_i^{n+1}-u_i^n}{\Delta t} + u_i^n\frac{u_i^n - u_{i-1}^n}{\Delta x} = 0 \qquad (3)\\ \frac{u_i^{n+1}-u_i^n}{\Delta t} + \frac{F_i^n - F_{i-1}^n}{\Delta x} = 0 \qquad (4)$$ Xt thm s bc hai MacCormack: $$u_i^* = u_i^n - \frac{\Delta t}{\Delta x} [F_{i+1}^n - F_i^n]\\ u_i^{n+1} = \frac{1}{2}(u_i^n + u_i^*) - \frac{\Delta t}{2\Delta x} [F_i^* - F_{i-1}^*] \qquad (5)$$

Tm nghim bi ton lan truyn sng xung kch c gii thiu bi 5, phn 1: $$u(x,0)=a,xx_0;u(x,0)=b,x>x_0;a>b$$ Nghim chnh xc: $$u(x,t)=a,xx_0+st;u(x,t)=b,x>x_0+st$$ vi s l vn tc dch chuyn gin on v bng (a+b)/2. Nh bit c nghim chnh xc, ta c th d dng kim tra kt qu tnh ton.

import numpy as np
import matplotlib.pyplot as plt

#chia li trn on [0, 1]
nx = 41
x, dx = np.linspace(0, 1, nx, retstep=True)

#bc thi gian
r = 0.4 #iu kin n nh: u_max dt/dx <= 1
dt = r*dx

#iu kin ban u
uL = 1.2
uR = 0.4

u0 = np.zeros(nx)
i = int(0.5/dx)
u0[:i] = uL
u0[i:] = uR

#s  sai phn 3
def nonconservative_scheme(nt):
    u = u0.copy()
    un = np.zeros(nx)
    for n in range(nt):
        un = u.copy()
        u[1:] = un[1:] - dt/dx*un[1:]*(un[1:] - un[:-1])
    return u

#s  sai phn 4 (F=u^2/2)
def conservative_scheme(nt):
    u = u0.copy()
    un = np.zeros(nx)
    for n in range(nt):
        un = u.copy()
        u[1:] = un[1:] - dt/(2*dx)*(un[1:]**2 - un[:-1]**2)
    return u

#s  bc hai MacCormack
def second_order_scheme(nt):
    u = u0.copy()
    u1 = u0.copy()
    un = np.zeros(nx)
    for n in range(nt):
        un = u.copy()
        u1[1:-1] = un[1:-1] - dt/(2*dx)*(un[2:]**2 - un[1:-1]**2)
        u[1:-1] = (un[1:-1] + u1[1:-1])/2 - dt/(4*dx)*(u1[1:-1]**2 - u1[:-2]**2)
    return u    

#so snh nghim tm c
nt = 25
u_1 = nonconservative_scheme(nt)
u_2 = conservative_scheme(nt)
u_3 = second_order_scheme(nt)
#nghim chnh xc: vn tc lan truyn sng xung kch s=(uL+uR)/2. = 0.8
u_exact = np.zeros(nx)
i = int((0.5 + 0.8*nt*dt)/dx)
u_exact[:i] = uL
u_exact[i:] = uR

plt.plot(x, u_1, x, u_2, x, u_3, x, u_exact)
plt.legend(['nonconservative scheme', 'conservative scheme', 'second-order scheme', 'exact solution'])
plt.show()

2. Phương pháp Godunov¶

S dng phng php th tch hu hn ta thu c s sai phn: $$\frac{du}{dt} + \frac{F_{i+1/2} - F_{i-1/2}}{\Delta x} = 0$$ vi xp x thi gian upwind bc 1 ta c: $$\frac{u_i^{n+1} - u_i^n}{\Delta t} + \frac{F_{i+1/2} - F_{i-1/2}}{\Delta x} = 0$$ y $u_i$ l gi tr trung bnh trn on [i-1/2, i+1/2]: $$u_i^n = \frac{1}{\Delta x}\int_{i-1/2}^{i+1/2} u(x, t_n) dx$$

Trong phng php Godunov hm dng F xc nh nh sau: $$F_{i-1/2}^* = F(u_{i-1/2}^*(u_{i-1/2}^L, u_{i-1/2}^R)); \quad F_{i+1/2}^* = F(u_{i+1/2}^*(u_{i+1/2}^L, u_{i+1/2}^R))$$

vi $u^*$ l nghim ca bi ton Riemann - phn r gin on trn b mt th tch hu hn: cho gi tr u ti hai bn tri phi ca b mt l uL, uR, tm nghim $u^*(u_L, u_R)$ trn b mt. Godunov ly uL, uR chnh l gi tr trung bnh u hai th tch hu hn hai bn b mt ang xt $u_{i+1/2}^*(u_i, u_{i+1})$.

Bi ton Riemann

Chng ta xem xt qua bi ton Riemann phn mt, by gi ta s nghin cu c th hn. Gi thit bi ton: $$u_t + F(u)_x = 0 \qquad (6)\\ u(t=0)=u_0 = \begin{cases} u_L; \quad x<0 \\ u_R; \quad x>0 \end{cases} $$

gii bi ton ny ta cn bit v ng c trng v sng.

ng c trng

Xt phng trnh $u_t + au_x = 0 \qquad (7)$.

ng c trng c nh ngha l ng trong khng gian m dc theo n phng trnh o hm ring (PDE) tr thnh phng trnh vi phn thng (ODE).

Xem $x=x(t)$, ta c: $$\frac{du}{dt} = \frac{\partial u}{\partial t} + \frac{\partial u}{\partial x} \frac{dx}{dt}$$ Nu $x'=a$ ta c: $$\frac{du}{dt} = \frac{\partial u}{\partial t} + a\frac{\partial u}{\partial x} = 0$$ Ngha l $u$ l hng s dc theo ng $\frac{dx}{dt} = a$. y $a$ c gi l vn tc c trng v bng dc ca ng $x(t)$. Trong trng hp tng qut $a=a(x,t)$.

Gi thit ti thi im ban u $x(t=0)=x_0$, ta c nghim phng trnh $x'=a$: $x=x_0+at$. Dc theo ng c trng u l hng s nn ta c: $$u(x,t) = u(x(0),0)=u_0(x_0)=u_0(x-at)$$ Nghim u chnh l $u_0$ ban u c dch chuyn vi vn tc $a$ sang tri hoc sang phi ty theo du ca $a$. Qu trnh ny ta gi l dch chuyn sng.

Quay li vi bi ton Riemann, phng trnh (6) c th vit li dng: $$u_t + Au_x = 0$$ vi $A=\frac{\partial F}{\partial u}$ gi l ma trn Jacobian. Gi tr ring ca ma trn A chnh l vn tc ng c trng hay vn tc dch chuyn sng. i vi phng trnh trnh (7): $A=|a|, \lambda = a$. Vi a l hng s, sng dch chuyn vi vn tc khng i nn hnh dng khng i. i vi phng trnh (2): $A=|u|, \lambda=u$, sng dch chuyn vi vn tc ph thuc thi gian v khng gian iu ny c th dn n s giao ct, phn tch ng c trng lm xut hin sng va p v sng gin nh gii thiu phn 1.

Trong trng hp bi ton Riemann cho phng trnh (2), ta thy sng hai bn tri phi chuyn ng vi hai vn tc khc nhau uL v uR.

Ti $x_{i+i/2}$ 'cu hnh' ng c trng c nhiu dng ph thuc mi lin h uL, uR, v d:

Hnh 1 ngoi cng bn tri: $u_L<0<u_R$, hnh 2:$u_R>u_L>0$, hnh 3: $u_L<u_R<0$, hnh 4 ngoi cng bn phi: $u_L>0>u_R, (u_L+u_R)/2<0$. Ch :$u=dx/dt$.

Cn c vo v tr $x_{i+1/2}$ thuc vng 1, 2 hay 3 v mi lin h uL, uR - sng no dch chuyn nhanh hn, v pha no ta xc nh $u^*$. V d nh trn hnh, hnh 1 - $x_{i+1/2}$ nm gia (1) v (2) - ly bng 0, hnh 2 - nm trong (1) - ly uL, hnh 3 - nm trong (2) - ly uR, hnh 4 - nm trong vng giao thoa, y ta ly sng no chuyn ng nhanh hn - nh hng mnh hn, chy ti ${i+1/2}$ nhanh hn. Tng hp tt c cc trng hp ta c:

Nu $u_L > u_R$ - sng xung kch: $$ u* = \begin{cases} u_L; \ if\quad \frac{u_L+u_R}{2}>0 \\ u_R; \ in\ other\ case \end{cases} $$ Nu $u_L \leq u_R$ - sng gin:

$$ u* = \begin{cases} u_L; \ if\quad u_L>0 \\ u_R; \ if\quad u_R<0 \\ 0; \quad if\quad u_L \leq 0 \leq u_R \end{cases} $$

#tm nghim bi ton Riemann
def decay_godunov(uL, uR):
    ustar = 0
    if uL > uR:
        if (uL + uR)/2. > 0: ustar = uL
        else: ustar = uR
    else:
        if uL > 0: ustar = uL
        elif uR < 0: ustar = uR
        else: ustar = 0
    return ustar

#tnh hm dng F(u*)
def flux(u):
    return u**2/2.

#s  Godunov
def godunov_method(nt):
    u = u0.copy()
    un = np.zeros(nx)
    for n in range(nt):
        un = u.copy()
        for i in range(1, nx-1):
            u_im05 = decay_godunov(un[i-1], un[i]) #u_{i-1/2}
            u_ip05 = decay_godunov(un[i], un[i+1]) #u_{i+1/2}
            u[i] = un[i] - dt/dx*(flux(u_ip05) - flux(u_im05))
    return u

#c th vit li code nh sau
def godunov_scheme(nt):
    u = u0.copy()
    un = np.zeros(nx)
    for n in range(nt):
        un = u.copy()
        ustar = np.array([decay_godunov(ul, ur) for ul, ur in zip(un[:-1], un[1:])])
        u[1:-1] = un[1:-1] - dt/dx*(flux(ustar[1:]) - flux(ustar[:-1]))
            
    return u
